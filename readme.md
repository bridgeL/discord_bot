# discord bot

欢迎任何人贡献插件到plugins文件夹！

## how to run

require python 3.8 or higher version

### get a token

https://www.writebots.com/discord-bot-token/

### install & run

1. git clone https://github.com/bridgeL/discord_bot.git
2. pip install poetry
3. poetry install
4. poetry shell
5. python run.py
